## 🌟 Overview

**Leh-Lu-Kya** (The Kuli App) connects customers with reliable, verified workers for on-demand tasks like carrying luggage, medical assistance, elderly care, and cargo transport. Built with modern web technologies, it offers a seamless experience for both customers and workers.

## ✨ Features

- **📱 User & Worker Portals**: Dedicated interfaces for customers and "Kulis" (workers).
- **🔒 Secure Verification**: robust ID verification and system status tracking.
- **🗺️ Real-Time Location**: Integrated with Leaflet for accurate mapping.
- **💳 Seamless Payments**: Powered by Stripe for secure transactions.
- **⚡ Lightning Fast**: Built on Vite + Vue 3.

## 🛠️ Tech Stack

- **Frontend**: Vue 3 (Composition API, `<script setup>`), Tailwind CSS v4, Lucide Icons
- **State Management**: Pinia
- **Routing**: Vue Router
- **Backend/BaaS**: Firebase (Authentication, Firestore, Storage)
- **Maps**: Leaflet
- **Payments**: Stripe

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Helix-1716/Leh-Lu-Kya.git
   cd "Kuli app"
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file in the root directory and add your Firebase and Stripe keys.

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Build for Production**
   ```bash
   npm run build
   ```
